Project
Missing Children Aging Prediction via FADING & GOAE
Mingyu Kim, Hyeongene Kim, Jungsoo Yoon, Gunwoo Kim

DSL Modeling Project, 2024 Fall
Code / Video

Based on FADING, the diffusion based model for aging and GOAE, the 3D rendering model with only one picture, we made two-step face aging architecture especially for Korean missing children. The addressed pre-process techniques such as upscaling and labeling and the results from this project are expected to contribute to related research for face aging.

Face Mosaic using Mask R-CNN
Mingyu Kim, Dongyoon Kim, Sooran Kim, Minwoo Park

YAI Toy Project, 2024 Summer
Code

Using the Mask R-CNN architecture for face detection, we made the model that applies mosaic effect for all detected faces Fine tuned by the WIDER FACE dataset, it performed well both for image and video. Although it was conducted as simple project for reviewing, this project helped me to broaden the technique related to AI research.
